Using the Calculator

Step 1: Enter Your Birth Date

Select your birth date using the date picker. This is the most important piece of information and is required for all calculations.

Step 2: Enter Your Birth Time

Enter your birth time as accurately as possible. The time affects:

  • Your Rising sign (Ascendant) - changes approximately every 2 hours
  • Moon position - moves about 12Â° per day
  • House positions in astrology
  • Human Design gates (especially personality vs design)

If you don't know your exact birth time, the default of 12:00 noon will give reasonable planetary positions, but your Rising sign and some Human Design calculations may be inaccurate.

Step 3: Enter Your Birth Location

Start typing your birth city and select from the autocomplete suggestions. This provides:

  • Accurate latitude and longitude for Rising sign calculation
  • Automatic timezone estimation
  • Midheaven (MC) calculation

Alternatively, click "Enter coordinates manually" to input exact latitude, longitude, and timezone offset.

Understanding Your Results

Western Astrology

The astrology section shows your complete natal chart:

  • Big Three - Sun (identity), Moon (emotions), Rising (appearance)
  • Element Balance - Fire, Earth, Air, Water distribution
  • Modality Balance - Cardinal, Fixed, Mutable distribution
  • Planetary Positions - All planets with sign and degree
  • Lunar Nodes - Life purpose indicators

Human Design

The bodygraph visualization shows your energy centers and channels:

  • Colored shapes - Defined (consistent) centers
  • White shapes - Undefined (receptive) centers
  • Black gates - Personality (conscious) activations
  • Red gates - Design (unconscious) activations
  • Channels - Connections between centers

Gene Keys

The Gene Keys section shows your Golden Path with three sequences:

  • Activation Sequence - Your genius and core gifts
  • Venus Sequence - Your relationship patterns
  • Pearl Sequence - Your prosperity and contribution

Each sphere shows a key number with its gift. The spectrum at the bottom shows the shadow-gift-siddhi transformation of your Life's Work key.

Developer API

All StarChart calculators can be imported and used directly in your JavaScript projects: 

import calculateAstrology from './src/calculators/astrology.js';
import calculateHumanDesign, { calculateGeneKeys } from './src/calculators/humandesign.js';

// Calculate Western Astrology
const astrology = calculateAstrology(
  '1990-06-15',  // birth date (YYYY-MM-DD)
  14.5,          // birth hour (decimal, 14.5 = 2:30 PM)
  -5,            // timezone (UTC offset)
  40.7128,       // latitude
  -74.0060       // longitude
);

// Calculate Human Design
const hd = calculateHumanDesign('1990-06-15', 14.5, -5);

// Calculate Gene Keys
const gk = calculateGeneKeys(hd);

Astrology Return Value

  • sun, moon, rising - Big three with sign, degree, longitude
  • planets - Mercury through Pluto
  • nodes - North and South lunar nodes
  • midheaven - MC position (if location provided)
  • balance - Element and modality counts
  • bigThree - Formatted string

Human Design Return Value

  • type - Name, strategy, signature, not-self
  • authority - Decision-making authority
  • profile - Numbers and description
  • incarnationCross - Name and gates
  • centers - Defined and undefined
  • gates - Personality and design activations
  • channels - Complete channel list

Gene Keys Return Value

  • activationSequence - Life's Work, Evolution, Radiance, Purpose
  • venusSequence - Attraction, IQ, EQ, SQ
  • pearlSequence - Vocation, Culture, Pearl
  • Each sphere includes key, line, shadow, gift, siddhi

Installation

# Clone the repository
git clone https://github.com/unforced/starchart.git
cd starchart

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
