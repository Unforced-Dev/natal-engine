What is MCP?

The Model Context Protocol (MCP) is an open standard that allows AI assistants to connect to external tools and data sources. By adding the StarChart MCP server, your AI can calculate birth charts on demand.

Quick Start

1. Clone the Repository

git clone https://github.com/unforced/starchart.git
cd starchart/mcp-server
npm install

2. Configure Claude Code

Add to your Claude Code settings file (~/.claude/settings.json):

{
  "mcpServers": {
    "starchart": {
      "command": "node",
      "args": ["/path/to/starchart/mcp-server/src/index.js"]
    }
  }
}

Replace /path/to/starchart with the actual path where you cloned the repository.

3. Restart Claude Code

After saving the settings, restart Claude Code to load the MCP server.

4. Try It Out

Ask Claude something like:

"Calculate my birth chart for June 15, 1990 at 2:30 PM in New York City"

Available Tools

calculate_starchart

The comprehensive tool that returns data from all three wisdom systems in one call.

  • birth_date (required) - Date in YYYY-MM-DD format
  • birth_time - Time in HH:MM format (24-hour)
  • latitude - Birth location latitude
  • longitude - Birth location longitude
  • timezone - UTC offset (auto-estimated if not provided)

calculate_astrology

Western natal chart with Sun, Moon, Rising, all planets, and aspects.

calculate_human_design

Human Design chart with Type, Strategy, Authority, Profile, Gates, and Channels.

calculate_gene_keys

Gene Keys profile with all three sequences (Activation, Venus, Pearl).

get_planetary_positions

Raw astronomical data for all celestial bodies.

How It Works

When you ask Claude about birth charts, it uses the StarChart MCP tools to:

  1. Parse your natural language query for birth data
  2. Call the appropriate calculation tool(s)
  3. Receive the structured chart data
  4. Format a helpful response with your results

All calculations happen locally on your machine using the same high-precision astronomy engine as the web calculator. No birth data is sent to external servers.

Troubleshooting

Server not found

Make sure the path in your settings.json is correct and absolute (starts with /). Check that you've run npm install in the mcp-server directory.

Tools not appearing

Try restarting Claude Code completely. You can verify the server works by running it directly: 

node /path/to/starchart/mcp-server/src/index.js

You should see "StarChart MCP Server running on stdio" in the console.

Calculation errors

Ensure dates are in YYYY-MM-DD format and times are in 24-hour HH:MM format. The birth_date parameter is required for all tools.
